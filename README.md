# Dorm Energy Simulator

Имитация и мониторинг энергопотребления в студенческом общежитии

## О проекте

Цель — создать систему, которая:
- генерирует реалистичные временные ряды энергопотребления (мощность, температура, влажность, освещённость и т.д.)
- обнаруживает аномалии двумя способами:
  - простые пороговые правила
  - обученная ML-модель (формат ONNX)
- отправляет понятные уведомления в Telegram при обнаружении проблем

Проект реализуется как консольное приложение (CLI) на C++ с использованием CMake.

## Текущий статус (Iteration 1 — Skeleton)

- Минимальный запускаемый CLI
- Парсинг аргументов командной строки
- Вывод версии
