## Краткое описание проекта

Проект реализует **прототип системы мониторинга энергопотребления** студенческого общежития с элементами раннего обнаружения проблемных ситуаций.

### Основные возможности текущей версии:
- ⚡ генерация реалистичных почасовых данных энергопотребления (имитация поведения реального общежития)
- 🔍 обнаружение аномалий двумя независимыми методами:
  - простые пороговые правила (перегрузка, аномально низкое ночное потребление)
  - инференс обученной модели машинного обучения в формате ONNX
- 📱 отправка понятных уведомлений в Telegram при обнаружении аномалий
- 📊 экспорт данных и результатов анализа в CSV

### Технологический стек:
- **C++20** — высокопроизводительная обработка данных
- **Python 3.11+** — обучение ML-моделей (исследования)
- **ONNX Runtime** — выполнение ML-моделей в C++
- **CLI11** — интерфейс командной строки

> 📌 **Статус:** Проект находится на стадии активной разработки (Iteration 1–2).  
> Текущая реализация работает с синтетическими данными, но архитектура спроектирована с прицелом на подключение реальных IoT-датчиков в будущем.
